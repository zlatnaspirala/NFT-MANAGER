### Install
```js
yarn init -y
yarn add hardhat -D 
npx hardhat
yarn add @openzeppelin/contracts  @openzeppelin/hardhat-upgrades
yarn add -D @nomiclabs/hardhat-ethers ethers @nomiclabs/hardhat-waffle ethereum-waffle chai
```

### Compile
```js
yarn compile 
```
